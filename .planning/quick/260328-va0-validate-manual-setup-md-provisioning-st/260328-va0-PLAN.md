## Tasks

### Task 1: Query Azure Resource State (all 10 steps)

Run the following `az` CLI commands (read-only) to check each step. All commands are safe ŌĆö no mutations.

**Step 1 ŌĆö API Gateway Env Vars:**
- `az containerapp show --name ca-api-gateway-prod --resource-group rg-aap-prod --query "properties.template.containers[0].env"` ŌĆö check if `AZURE_PROJECT_ENDPOINT`, `ORCHESTRATOR_AGENT_ID`, `AZURE_CLIENT_ID`, `AZURE_TENANT_ID`, `CORS_ALLOWED_ORIGINS` are set

**Step 2 ŌĆö Foundry RBAC:**
- Get gateway principal: `az containerapp show --name ca-api-gateway-prod --resource-group rg-aap-prod --query "identity.principalId" -o tsv`
- Check role assignments: `az role assignment list --assignee <principalId> --query "[?roleDefinitionName=='Azure AI Developer']"`

**Step 3 ŌĆö Log Analytics on Web UI:**
- `az containerapp show --name ca-web-ui-prod --resource-group rg-aap-prod --query "properties.template.containers[0].env"` ŌĆö check `LOG_ANALYTICS_WORKSPACE_ID`

**Step 4 ŌĆö Cosmos DB `incidents` container:**
- Get Cosmos account: `az cosmosdb list --resource-group rg-aap-prod --query "[0].name" -o tsv`
- Check container: `az cosmosdb sql container show --account-name <name> --database-name aap --name incidents --resource-group rg-aap-prod`
- Check Cosmos SQL role assignments: `az cosmosdb sql role assignment list --account-name <name> --resource-group rg-aap-prod`

**Step 5 ŌĆö Teams Bot Registration:**
- `az bot show --name aap-teams-bot-prod --resource-group rg-aap-prod` ŌĆö check if Azure Bot resource exists
- `az containerapp show --name ca-teams-bot-prod --resource-group rg-aap-prod` ŌĆö check if Container App exists

**Step 6 ŌĆö GitHub Actions Secrets/Variables:**
- `gh secret list` + `gh variable list` ŌĆö check which secrets and variables are configured

**Step 7 ŌĆö Runbook Seeding:**
- Cannot verify remotely without PG access. Check if seed script exists locally and whether the Container App has POSTGRES_HOST set.

**Step 8 ŌĆö Multi-Subscription Reader:**
- `az role assignment list --assignee "65cf695c-1def-48ba-96af-d968218c90ba" --all --query "[?roleDefinitionName=='Reader']"` ŌĆö check if Reader on multiple subscriptions

**Step 9 ŌĆö Entra Redirect URIs:**
- `az ad app list --filter "displayName eq 'aap-web-ui-prod'" --query "[0].spa.redirectUris"` ŌĆö or use known app ID

**Step 10 ŌĆö Secret Rotation:**
- Check if `credentials.tfvars` is in `.gitignore`: local file check
- This is an operational hygiene step ŌĆö flag as manual review needed
