# Quick Task: Fix ObservabilityTab bugs + UI modernisation

## Context

The 07-UI-REVIEW.md audit (score 20/24) identified two priority bugs and the UI currently has a functional but flat visual style. This task fixes the two contract deviations and applies a targeted visual polish pass across the dashboard using only Fluent UI v9 primitives.
