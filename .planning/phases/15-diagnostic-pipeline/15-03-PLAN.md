# Plan 15-03: Enrich IncidentSummary Model

## Objective

Add `resource_name`, `resource_group`, `resource_type`, `investigation_status`, and `evidence_collected_at` fields to `IncidentSummary`. Update `incidents_list.py` to populate these fields from the Cosmos incident documents. This is required for the UI to show which resource an alert affects and to drive the "Investigate" CTA.

## Background

`IncidentSummary` in `services/api-gateway/models.py` currently has:
```python
class IncidentSummary(BaseModel):
    incident_id: str
    severity: str
    domain: str
    status: str
    created_at: str
    title: Optional[str] = None
    resource_id: Optional[str] = None
    subscription_id: Optional[str] = None
```

The ARM resource_id (`/subscriptions/sub/resourceGroups/rg/providers/type/name`) contains `resource_name` and `resource_group` but the UI has no parsed versions. `investigation_status` is needed so the UI knows whether evidence is ready.

### Task 2: Add helper `_parse_resource_id` to `services/api-gateway/incidents_list.py`

```python
def _parse_resource_id(resource_id: Optional[str]) -> dict:
    """Extract name, resource_group, resource_type, subscription_id from ARM resource ID.

    ARM resource ID format:
    /subscriptions/{sub}/resourceGroups/{rg}/providers/{namespace}/{type}/{name}

    Returns dict with keys: resource_name, resource_group, resource_type, subscription_id.
    All values are None if resource_id is None or malformed.
    """
    if not resource_id:
        return {"resource_name": None, "resource_group": None, "resource_type": None, "subscription_id": None}

    parts = resource_id.split("/")
    # Normalize to lowercase for comparison, but extract original-case values
    lower = resource_id.lower().split("/")

    result = {"resource_name": None, "resource_group": None, "resource_type": None, "subscription_id": None}

    try:
        if "subscriptions" in lower:
            idx = lower.index("subscriptions")
            result["subscription_id"] = parts[idx + 1]
        if "resourcegroups" in lower:
            idx = lower.index("resourcegroups")
            result["resource_group"] = parts[idx + 1]
        if "providers" in lower:
            idx = lower.index("providers")
            # type = namespace/type, name = parts[idx+3]
            if idx + 3 < len(parts):
                result["resource_type"] = f"{lower[idx+1]}/{lower[idx+2]}"
                result["resource_name"] = parts[idx + 3]
            elif idx + 2 < len(parts):
                result["resource_type"] = lower[idx + 1]
        # Fallback: last non-empty segment is the resource name
        if not result["resource_name"]:
            non_empty = [p for p in parts if p]
            if non_empty:
                result["resource_name"] = non_empty[-1]
    except (IndexError, ValueError):
        pass

    return result
```

### Task 5: Unit tests for `_parse_resource_id`

**File:** `services/api-gateway/tests/test_incidents_list.py` (add test cases)

```python
from services.api_gateway.incidents_list import _parse_resource_id

def test_parse_resource_id_vm():
    rid = "/subscriptions/sub-123/resourceGroups/rg-prod/providers/Microsoft.Compute/virtualMachines/vm-prod-001"
    result = _parse_resource_id(rid)
    assert result["subscription_id"] == "sub-123"
    assert result["resource_group"] == "rg-prod"
    assert result["resource_type"] == "microsoft.compute/virtualmachines"
    assert result["resource_name"] == "vm-prod-001"

def test_parse_resource_id_storage():
    rid = "/subscriptions/abc/resourceGroups/rg-storage/providers/Microsoft.Storage/storageAccounts/stgprod001"
    result = _parse_resource_id(rid)
    assert result["resource_name"] == "stgprod001"
    assert result["resource_type"] == "microsoft.storage/storageaccounts"

def test_parse_resource_id_none():
    result = _parse_resource_id(None)
    assert result == {"resource_name": None, "resource_group": None, "resource_type": None, "subscription_id": None}

def test_parse_resource_id_malformed():
    result = _parse_resource_id("/invalid/path")
    assert result["resource_name"] == "path"  # fallback: last segment

def test_incident_summary_has_new_fields():
    from services.api_gateway.models import IncidentSummary
    s = IncidentSummary(
        incident_id="inc-001",
        severity="Sev1",
        domain="compute",
        status="open",
        created_at="2026-04-01T10:00:00Z",
        resource_name="vm-prod-001",
        resource_group="rg-prod",
        resource_type="microsoft.compute/virtualmachines",
        investigation_status="evidence_ready",
    )
    assert s.resource_name == "vm-prod-001"
    assert s.investigation_status == "evidence_ready"
```
