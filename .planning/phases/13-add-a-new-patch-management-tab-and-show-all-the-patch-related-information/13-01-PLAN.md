## Goal

Add a Patch Management tab to the DashboardPanel in the web UI. This includes:
- Two new API gateway endpoints exposing ARG patch data (assessment + installations)
- Next.js proxy routes for the new gateway endpoints
- A PatchTab component with 5 summary cards and 2 data tables
- Wiring the tab into DashboardPanel.tsx
