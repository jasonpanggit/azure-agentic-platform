# fabric

Microsoft Fabric assets for the platform's real-time detection plane: KQL schemas, data retention policies, ingestion policies, and User Data Functions triggered by Fabric Activator.

## Contents

- `kql/` — KQL queries for Eventhouse (schemas, retention, ingestion policies, Activator trigger queries)
- `user-data-function/` — Python User Data Functions invoked by Activator to call the API gateway when detection rules fire
