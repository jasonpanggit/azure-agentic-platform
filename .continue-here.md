<decisions_made>

- **Use Foundry `connected_agent` tools for routing** (not HandoffOrchestrator)
  - Foundry's native connected_agent type enables direct agent-to-agent routing
  - Tool names must match `^[a-zA-Z_]+$` → underscores required (arc_agent not arc-agent)

- **Register tools on Foundry agent definitions** (not just container runtime)
  - When orchestrator routes via connected_agent, Foundry executes the target agent
  - The target agent's Foundry definition must have its function tools registered
  - Container runtime tools alone are not visible to Foundry's connected-agent execution

- **Foundry API version: `2025-05-01`**
  - Only version that works for the project endpoint
  - Versions 2025-04-01 and earlier return "API version not supported"
</decisions_made>
