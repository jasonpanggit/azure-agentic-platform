# docs

Project-level documentation covering platform architecture, agent specifications, operational guides, and the delivery roadmap.

## Contents

- `agents/` — per-agent specification documents (tools, prompts, handoff contracts)
- `ops/` — operational runbooks for platform engineers (detection plane, logging, RBAC setup)
- `roadmap/` — phase-by-phase delivery roadmap and milestone tracking
- `troubleshooting/` — common failure patterns and resolution guides
- `verification/` — verification checklists for each delivery phase
- `superpowers/` — documentation on platform "superpower" capabilities
- `observability-wiring.md` — how OpenTelemetry traces flow from agents through App Insights
- `BOOTSTRAP.md` — step-by-step platform bootstrap guide for a new environment
- `MANUAL-SETUP.md` — manual setup steps that cannot be automated by Terraform
